How to Use This Tutorial

Tutorial Panel (Left)

  • Follow the steps in order — each step builds on the previous one
  • Click Continue after completing each step to advance
  • Click Show Me to auto-type the commands for the current step
  • Code blocks can be clicked to copy commands

Cisco IOS XE Simulator (Right)

  • Type Cisco IOS commands directly in the terminal
  • Switch between R1, R2, PC1, PC2 using the tabs
  • Commands are auto-completed — press Tab to complete
  • Use ↑ / ↓ to recall command history
  • The ? key shows available commands in any mode

Network Topology

  • Click on a router in the topology to switch to its terminal
  • Drag the divider below the topology to resize it
  • Hover over labels to see interface details

Toolbar

  • A A A — Change terminal font size (small / medium / large)
  • Reset All — Reset all routers and PCs to factory defaults
  • Feedback tab (right edge) — Report bugs or suggestions

Achievement Badge

  • Complete all 14 steps to earn the Quantum Safe Networking Pioneer badge
  • Gold badge — complete all steps by typing commands yourself (no "Show Me")
  • Silver badge — complete with some "Show Me" assistance
  • Download your badge as PNG or SVG and share it on LinkedIn
  • The badge includes a link back to this tutorial for others to discover

Tips

  • Every show command output line is clickable for an explanation
  • Command keywords are linked to official Cisco documentation
  • The tunnel visualization appears automatically when the VPN is established
